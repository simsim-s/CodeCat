CSS3 Additional Selectors

Table
Pseudo-classes
Selector Meaning
1. a:link{} styles for non-visited links
2. a:visited{} styles for viseted links
3. :hover{} an element which is hovered with a mouse
4. :focus{} an element which a user is ready to interact with
5. :active{} an element which is activated by a user
Attribute Selectors
Selector Meaning
1. [] any element with a certain name attribute, for example -class- attribute: p[class]
2. [=] any element with a certain value of attribute, e.g.:p[class="dog"]
3. [~=] an element whose class is a list of space-seperated words, one of which is "dog": p[class~="dog"]
4. [^=] an element with an attribute whose value begins with, for example "d":p[attr^="d"]
5. [*=] an element whose attribute contains, for example letters "do": p[attr*="do"]
6. [$=] any element with an attribute whose value ends with a letter "g":p[attr$="g"]
